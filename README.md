# Лабораторные работы по PHP

Этот репозиторий содержит лабораторные работы по PHP:

- Сравнение типов данных в PHP
- Работа с базой данных MySQL
- Создание, чтение, обновление, удаление записей (CRUD)
- Работа с формами и пользовательскими параметрами

## Файлы проекта:
- index.php - главная страница со сравнением типов данных
- get.php - вывод данных из базы данных  
- add.php - добавление новых записей в БД
- delete.php - удаление записей по ID

## Технологии:
- PHP 8.4
- MySQL
- PDO
- HTML/CSS
